# 🎬 Movie Recommendation System

## 🚀 Overview
This project is a genre-based movie recommendation system that suggests movies based on user preferences. It is built using machine learning techniques and deployed using Streamlit for an interactive user interface.

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## 🎯 Features
- Recommends movies based on selected genre  
- Processes movie dataset for similarity-based suggestions  
- Interactive web interface using Streamlit  
- Fast and real-time recommendations  

---

## 🧠 How It Works
1. Load movie dataset  
2. Preprocess data (cleaning & transformation)  
3. Apply similarity-based algorithm  
4. Recommend movies based on user input
